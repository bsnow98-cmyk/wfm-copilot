## Sending notes

- **Don't BCC them all.** One personalized send each, ideally over 30-60 minutes (reduces spam-flag risk).
- **From a real Gmail or Resend domain**, not a no-reply.
- **Track replies in a spreadsheet:** Name | Replied? | Pain mentioned | Tool mentioned | Follow-up?
- **Respond fast.** If anyone replies, get back in <2 hours. Discovery momentum dies overnight.
- **The signal you're looking for:** 3+ unprompted name the same pain (e.g. "scheduling rebuilds when someone calls out sick") = wedge confirmed.

## What success looks like

- 1-2 replies in 5 sends = normal cold-outbound rate. That's enough to learn from.
- 0 replies in 7 days = pull the next 25 from Vibe Prospecting (cost: ~75 credits) and try a different angle.
- 3+ substantive replies = book Zoom calls, validate the pain in conversation, THEN decide on Phases 3-7.
